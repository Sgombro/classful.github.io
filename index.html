Classful Addressing Overview

In classful addressing, IP addresses are divided into five classes: A, B, C, D, and E. Here is a quick summary:

  • Class A: Range 0.0.0.0 - 127.255.255.255, Default Subnet Mask: 255.0.0.0
  • Class B: Range 128.0.0.0 - 191.255.255.255, Default Subnet Mask: 255.255.0.0
  • Class C: Range 192.0.0.0 - 223.255.255.255, Default Subnet Mask: 255.255.255.0
  • Class D: Range 224.0.0.0 - 239.255.255.255 (Multicast)
  • Class E: Range 240.0.0.0 - 255.255.255.255 (Experimental)
